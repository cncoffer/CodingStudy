# 8-18

题目要求不能使用乘法除法和mod运算符, 那就只能用减法了.

# 算法一

首先将两个数都取绝对值, 并记录下符号.
然后不停的做差, 直到减成负数.
然后`做差的次数-1`即为商.

但是不停的做差是比较慢的, 因为如果是下面这样的输入, 就要做`2147483647`次差值计算.
    2147483647
    1

# 算法二

想不到该怎么办了, 找了答案.
原来还有一种运算可以使用, 那就是移位, 因为任意数都可以表示为
    `a = a0*2^0 + a1*2^1 + ... + an*2^n`

所以可以通过移动除数, 不停的减掉刚好小于被除数的除数的移位结果.
